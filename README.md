# Food Delivery Analysis

This project analyzes food delivery data to understand order patterns, delivery time, customer behavior, and restaurant performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Order volume trends
- Delivery time distribution
- Top restaurants
- Revenue analysis
- Customer behavior patterns

## Outcome
The project provides insights to optimize delivery operations and improve customer satisfaction.
